Summary

The purpose of this project was to analyze the R diamonds dataset. To accomplish this analysis, I first accessed the data from R Studio which comes pre-installed with R: The R Project for Statistical Computing.

The ultimate goal was to build a predictive model of diamonds that was going to help figure out whether a given diamond is a good deal or not. For those looking to buy, this dataset can help you understand what goes into the price of a diamond.
